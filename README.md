# PointBlank
## Dependencies
Part of this project is written in python3. Use pip3 to install libraries.  
On OS X, please run
```
pip3 install pyobjc-core
pip3 install pyobjc
```
On Linux, please run
```
pip3 install python3-Xlib
```
For all platforms, the following python libraries are also needed:
```
PyQt5
pyautogui
```
